# AKKA HTTP STARTER #

Scaffolding/starter project for akka-http 

## Project Stack ##

- Scala: 2.12.2
- Akka: 2.5.3
- Akka-http: 10.0.9
- Circe: 0.8.0 for JSON Marshalling/UnMarshalling
- Reactive Kafka: 0.16
- Scalatest: 3.0.1
- Logstash: 4.11
- Swagger

## Usage ##
Start server:
```
sbt run
```
For swagger page:
```
http://localhost:8111
```
